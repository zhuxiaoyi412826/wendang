# Markdown文档阅读网站

这是一个基于Vue.js的Markdown文档阅读网站，用于展示和阅读javaweb系列的技术文档。

## 功能特点

- 支持Markdown文件的渲染和阅读
- 提供文章列表导航
- 自动生成文章大纲
- 支持亮色/暗色模式切换
- 代码块支持复制和全屏显示

## 项目结构

- `index.html`: 网站主页面
- `app.js`: Vue应用主逻辑
- `styles.css`: 样式文件
- `resources/`: 存放Markdown文档的目录

## 使用方法

1. 克隆仓库到本地
2. 在浏览器中打开index.html文件
3. 点击左侧文章列表浏览不同的文档
